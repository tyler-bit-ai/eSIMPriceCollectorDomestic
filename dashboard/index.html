크롤링 시점 선택

상단 선택기에서 이전 publish 시점 데이터를 고를 수 있습니다. 선택을 바꾸면 화면 전체가 해당 시점 기준으로 다시 갱신됩니다.

Filters

사이트, 국가, 사용일수, 데이터, 망 유형, 정렬 기준으로 전체 비교 조건을 좁힙니다. 아래 모든 섹션에 공통 적용됩니다.

Price Distribution

국가별 일수 구간의 최저가 흐름을 비교합니다. 표시 국가와 표시 개수를 줄여 원하는 밴드만 볼 수 있고, 카드를 누르면 상세표 필터가 연결됩니다.

Local vs Roaming

같은 사이트의 local / roaming 가격 차이를 비교합니다. 사이트와 표시 개수를 조절해 필요한 카드만 집중해서 볼 수 있습니다.

Drill-down

현재 선택한 조건의 비교 행을 표로 보여줍니다. 우측 상단의 Excel 다운로드 버튼으로 현재 보이는 결과를 CSV 파일로 받을 수 있습니다.
