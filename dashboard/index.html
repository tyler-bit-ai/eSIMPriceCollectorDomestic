크롤링 시점 선택

상단 드롭다운에서 시점을 최대 2개까지 체크할 수 있습니다. 먼저 선택한 시점이 기준이 되며, 두 시점을 고르면 아래 시점별 변경 상품 섹션에 차이만 따로 표시됩니다.

Filters

사이트, 국가, 사용일수, 데이터, 망 유형, 정렬 기준으로 전체 비교 조건을 좁힙니다. 아래 모든 섹션에 공통 적용됩니다.

Price Distribution

국가별 일수 구간의 최저가 흐름을 비교합니다. 표시 국가와 표시 개수를 줄여 원하는 밴드만 볼 수 있고, 카드를 누르면 상세표 필터가 연결됩니다. 섹션 안의 드릴다운 해제 버튼으로 바로 원복할 수 있습니다.

Local vs Roaming

같은 사이트의 local / roaming 가격 차이를 비교합니다. 사이트와 표시 개수를 조절해 필요한 카드만 집중해서 볼 수 있고, 카드 클릭 후에도 섹션 안에서 바로 드릴다운 해제가 가능합니다.

Snapshot Diff

선택한 2개 시점 사이에서 가격이 달라졌거나 한쪽 시점에만 존재하는 상품만 따로 모아 보여줍니다. 현재 필터 조건이 그대로 적용됩니다.

Drill-down

현재 선택한 조건의 비교 행을 표로 보여줍니다. 시점 비교 정보는 별도 섹션에서 확인하고, 상세표와 Excel 다운로드는 현재 기준 시점 데이터 자체에 집중합니다.
